Desktop glyph icons
===================

Source: Microsoft "Fluent System Icons"
        https://github.com/microsoft/fluentui-system-icons
License: MIT License (commercial use permitted)

The .svg files in this directory are unmodified "48 filled" assets from the
Fluent System Icons set. They are rasterized at build time by
tools/svg2icons.py into kernel/include/desktop_icons.h (8-bit coverage masks),
which the MicroNT shell composites in theme colors onto desktop tiles.
